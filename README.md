# Simple Interest Calculator

## Description
This project is a simple interest calculator built using Bash scripting. It calculates the simple interest based on user input.

## Formula
Simple Interest = (Principal × Rate × Time) / 100

## Usage
Run the script and enter:
- Principal amount
- Rate of interest
- Time period

## Example
If Principal = 1000, Rate = 5%, Time = 2 years  
Simple Interest = 100
